イジングモデルは多数の素子から構成されるシステムである。各素子の状態を表す変数をsi は、± 1 の値のどちらかをとる。
そして、システムのエネルギーE は以下の形の相互作用の総和になっている。

E(S) = ∑i,j Jij×si×sj

ここで、Jijは相互作用の強さを決める相互作用係数である。
統計物理の理論によると、周りの環境とのエネルギーのやりとりがあると、
システムの状態S = fsig は環境の温度T によって支配される次のボルツマン分布に従う。

P(S) = exp(-E(S)/T)/Z

Z は規格化因子で、Z = Σ exp(－ E(S)/T。また、ボルツマン定数は簡単化の為に省略している。
したがって、状態変数による物理量A の期待値は以下の式で定義される。

A(T) = ∑ A(S)P(S) = ∑ A(S)exp(-E(S)/T)/T = <A(S)>

記号<.>は期待値を表す。

トーラス上の二次元強磁性イジングモデルJ = 1 を考える。システムサイズはL×Lとする。
ただし、トーラスの表面に貼付けたグリッドの格子点の座標を(x,y) (0 ≦ x,y ≦ (L － 1)) とし、
周期的境界条件は、格子点(L,y) = (0,y)、(x,L) = (x,0) とする。
以下の物理量、磁化m を考える。

m = |∑si|/N

Nはイジングスピン数である。
